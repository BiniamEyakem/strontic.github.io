    Console utility for management of Storage Spaces and Pools

Supported commands:

  Get-Pool                - Lists all the pools in the system
  Set-Pool                - Changes the properties of a pool
  New-Pool                - Creates a pool
  Remove-Pool             - Deletes a pool
  Refresh-Pool            - Refreshes the properties of a pool
  Upgrade-Pool            - Upgrades a pool
  Optimize-Pool           - Rebalances allocations in a pool
  RebalanceMetadata-Pool  - Rebalances a pool's metadata
  Get-Drive               - Lists all the drives in a pool
  Set-Drive               - Changes the properties of a drive
  Add-Drive               - Adds a drive to a pool
  Remove-Drive            - Removes a drive from a pool
  Reset-Drive             - Resets the state of a drive
  Refresh-Drive           - Refreshes the properties of a drive
  Retire-Drive            - Retires a drive
  Reallocate-Drive        - Reallocates extents off of a drive
  Resize-Drive            - Resizes a drive by extending its spaces partition
  SetMaintenance-Drive    - Sets maintenance mode on a drive
  Get-Enclosure           - Lists all the enclosures connected to the system
  Set-Enclosure           - Changes the properties of elements in an enclosure
  Refresh-Enclosure       - Refreshes the properties of an enclosure
  Get-Space               - Lists all the spaces in a pool
  Set-Space               - Changes the properties of a space
  New-Space               - Creates a space
  Remove-Space            - Deletes a space
  Repair-Space            - Repairs a space
  Attach-Space            - Attaches a space
  Detach-Space            - Detaches a space
  RebalanceMetadata-Space - Rebalances a space's metadata
  Snapshot-Space          - Create a snapshot of a space
  Unlink-Space            - Unlink a space from its snapshots
  Get-Tier                - Lists all the tiers in a pool
  Set-Tier                - Changes the properties of a tier
  New-Tier                - Creates a pool tier
  Remove-Tier             - Deletes a tier
  Get-Task                - Lists all the tasks in a pool
  Stop-Task               - Stops specified tasks in a pool
  Get-Control             - Lists all the control settings
  Set-Control             - Changes the control settings
  PauseRepair-Control     - Pause repair
  ResumeRepair-Control    - Resume repair
